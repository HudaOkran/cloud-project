الدرس الأول: أوامر Git الأساسية

هنا أهم أوامر نظام Git التي نستخدمها لإدارة نسخة المشروع محلياً:

  • git init: لتهيئة مجلد عادي ليصبح مستودع Git محلي.
  • git status: لعرض حالة الملفات (المعدلة، المضافة، غير المتتبعة).
  • git add .: لإضافة جميع الملفات إلى منطقة التخزين المؤقت (Staging Area).
  • git commit -m "رسالة": لحفظ التغييرات بشكل دائم محلياً باسم محدد.
  • git log: لعرض سجل جميع عمليات الحفظ (Commits).
