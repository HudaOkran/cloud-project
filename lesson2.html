الدرس الثاني: التعاون والرفع إلى GitHub

تستخدم هذه الأوامر للربط بين المستودع المحلي ومستودع GitHub البعيد:

  • git remote add origin <رابط>: لربط المستودع المحلي بالبعيد.
  • git push -u origin main: لرفع التغييرات المحلية إلى الفرع الرئيسي على GitHub.
  • git pull origin main: لسحب آخر التغييرات من GitHub إلى جهازك.
  • git clone <رابط>: لنسخ مستودع موجود على GitHub إلى الجهاز.
